REST API - JSON
RESTful API — it's a way for two applications to communicate over the internet using HTTP.
It uses standard HTTP methods:

GET — fetch data
POST — create data
PUT/PATCH — update data
DELETE — delete data



GET /users HTML DOCUMENT RENDOR
GET /api/users - List All users -DONE



GET   /api/users/1   get the user with ID 1    -DONE
GET   /api/users/2   get the user with ID 2     -DONE
 DYNAMIC PATH PARAMETRES
 GET     /api/users/:id 
    :id -> variable       | dynamic            -DONE


 
POST   /api/users   create new user               -DONE


PATCH   /api /user/1  edit the user with id 1             -DONE


DELETE   /api/user/1   delete the user with ID 1             -DONE
